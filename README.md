## Current Research

### Returns and Volatility Factor Structures in Chinese Commodity Futures

I am currently contributing to a research project that examines common return and volatility factor structures in Chinese commodity futures using five-minute market data from 2017 to 2024.

The project investigates whether commodity returns and volatility are driven by common market factors and explores the relationship between commodity-market volatility and broader financial-market conditions.
